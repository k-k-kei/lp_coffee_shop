# 作ろうと思ったきっかけ
自身が運営しているコーヒーブランドのLPを作りたかったためです。
<br>
友人にコーヒーショップをやっていると言ってもその雰囲気が伝わらないことが多く、
<br>
その課題を解決するためにブランドイメージが伝わるページを目指しました。

# こだわったポイント
スッキリした印象を得てもらうために見出しとメニューのジャンプ率を意識しました。
<br>
またコンセプトを伝える中段は、見る側の負荷を下げることを意識して中央寄せにしています。
<br>
また、情報をただ伝える時は静的に表示をして、ストーリーを伝える時はアニメーションを多く使って動的に表示させるなど、
<br>
サイト内でもメリハリをつけることを意識して制作しました。
